**최종 수정일: 2026년 3월 12일**

## 1. 수집하는 개인정보

**본 확장 프로그램은 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.**

## 2. 데이터 처리 방식

- 사용자가 업로드하는 CSV 파일은 **브라우저 내부(로컬)**에서만 처리됩니다.
- 모든 데이터 파싱 및 자동 배정 작업은 사용자의 컴퓨터에서만 수행됩니다.
- **외부 서버로 데이터가 전송되지 않습니다.**
- CSV 파일의 내용(학생 이름, 학년, 반, 번호, 동아리명 등)은 브라우저 메모리에서만 임시로 사용되며, 확장 프로그램을 닫으면 즉시 삭제됩니다.

## 3. 사용 권한

본 확장 프로그램은 다음 권한을 사용합니다:

| 권한 | 사용 목적 |
|---|---|
| activeTab | 현재 활성 탭의 나이스 페이지와 상호작용 |
| scripting | 나이스 페이지의 DOM 요소(드롭다운, 체크박스 등) 자동 조작 |
| sidePanel | 사이드바 UI 표시 |
| storage | 확장 프로그램 설정 저장 (로컬만) |
| downloads | CSV 템플릿 파일 다운로드 |

## 4. 호스트 권한

`*.neis.go.kr` 도메인에서만 동작하며, 해당 사이트의 동아리 부서배정 화면에서 DOM 조작을 수행합니다.

## 5. 제3자 라이브러리

- **PapaParse** (MIT License): CSV 파일 파싱에 사용됩니다. 로컬에서만 동작합니다.
