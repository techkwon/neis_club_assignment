- Name: 나이스 동아리 배정 도우미 (NEIS Club Assignment Helper)
- Summary: Automate NEIS club assignments with CSV files.
- Description: 
NEIS Club Assignment Helper is a browser extension designed to automate student assignments for the 4th generation NEIS (National Education Information System).

Instead of manually clicking hundreds of students, simply upload a CSV file and let the helper do the work.

Key Features:
- CSV-based automation: Fast and accurate student assignments.
- Smart Matching: Name-based matching with Student Number fallback.
- Sorting: Auto-sort data to minimize group changes.
- Security: All data is processed locally on-device. No data transmission to external servers.
