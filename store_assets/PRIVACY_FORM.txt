# 크롬 웹 스토어 - 개인정보 보호 양식 작성 가이드
# 아래 내용을 각 필드에 복사-붙여넣기 하세요.

===============================================
📌 전용 목적 설명
===============================================
4세대 나이스(NEIS) 동아리활동관리 > 부서배정 화면에서, 사용자가 업로드한 CSV 파일의 학생 데이터(학년, 반, 번호, 이름, 동아리명, 부서구분)를 기반으로 동아리 부서 배정 작업을 자동화하는 것이 이 확장 프로그램의 전용 목적입니다. 수백 명의 학생을 수동으로 하나씩 클릭하여 배정하는 시간 소모적인 반복 작업을 자동화하여 교사의 업무 효율을 높입니다. 모든 데이터는 브라우저 내부에서만 처리되며, 외부 서버로 전송되지 않습니다.

===============================================
📌 activeTab 사용 근거
===============================================
사용자가 현재 보고 있는 나이스(NEIS) 부서배정 페이지의 탭에 접근하여, 해당 페이지가 동아리 부서배정 화면인지 확인하고, 사이드바에서 배정 시작 명령을 보낼 때 활성 탭과 통신하기 위해 필요합니다. 나이스 사이트가 아닌 탭에서는 어떠한 동작도 수행하지 않습니다.

===============================================
📌 scripting 사용 근거
===============================================
나이스(NEIS) 부서배정 페이지의 DOM 요소(학년/반 드롭다운 선택, 학생 체크박스 클릭, 동아리 체크박스 클릭, 저장 버튼 클릭, 확인 대화상자 처리)를 프로그래밍 방식으로 조작하여 자동 배정을 수행하기 위해 필요합니다. content script를 통해 나이스 페이지의 eXbuilder6 기반 UI 요소들과 상호작용합니다.

===============================================
📌 storage 사용 근거
===============================================
확장 프로그램의 사용자 설정(자동 정렬 토글, 화면 꺼짐 방지 토글 등)을 로컬에 저장하기 위해 사용합니다. chrome.storage.local만 사용하며, 사용자의 개인 데이터나 CSV 데이터는 저장하지 않습니다. 모든 저장 데이터는 사용자의 브라우저 로컬에만 존재합니다.

===============================================
📌 downloads 사용 근거
===============================================
사용자가 CSV 템플릿 양식 파일(나이스_동아리_배정양식.csv)을 다운로드할 때 사용됩니다. 이 템플릿은 확장 프로그램 내에서 생성되는 정적 CSV 파일이며, 사용자가 올바른 형식으로 학생 데이터를 입력할 수 있도록 안내하는 용도입니다.

===============================================
📌 sidePanel 사용 근거
===============================================
나이스 부서배정 페이지 옆에 사이드바 UI를 표시하기 위해 사용합니다. 사이드바에는 CSV 파일 업로드, 배정 시작/일시정지/정지 버튼, 실시간 진행 상황, 배정 로그(성공/실패)가 표시됩니다. 사이드바를 사용함으로써 나이스 페이지를 가리지 않고 동시에 확인할 수 있습니다.

===============================================
📌 호스트 권한 사용 근거
===============================================
*.neis.go.kr 도메인에 대한 호스트 권한은 나이스(NEIS) 사이트의 동아리 부서배정 페이지에 content script를 삽입하기 위해 필요합니다. content script는 해당 페이지의 학년/반 드롭다운, 학생 목록 그리드, 동아리 목록 그리드, 저장 버튼 등 DOM 요소를 자동으로 조작하여 배정 작업을 수행합니다. 나이스 사이트 외의 다른 웹사이트에서는 어떠한 동작도 수행하지 않습니다.
