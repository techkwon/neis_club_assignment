===========================================
📌 상세 설명
===========================================
나이스 동아리 배정 도우미는 4세대 나이스(NEIS)의 동아리 부서배정 작업을 자동화하는 웨일 확장 프로그램입니다.

수백 명의 학생을 하나하나 클릭하여 동아리에 배정하는 반복 작업을 CSV 파일 하나로 해결하세요. 현직 선생님들의 업무 효율을 획기적으로 높여드립니다.

✨ 주요 기능
• CSV 기반 자동 배정: 학년/반/번호/이름/동아리명이 담긴 CSV 업로드 시 자동 처리
• 스마트 매칭: 이름 우선 검색, 실패 시 출석번호 폴백 지원
• 부서구분 매칭: 동명 동아리 구분 (동아리활동/자율동아리/청소년단체/학교스포츠클럽)
• 최적화 정렬: 학년·반·번호 순 정렬로 페이지 전환 최소화
• 화면 꺼짐 방지: 장시간 작업 시 시스템 절전 모드 자동 차단
• 🔒 로컬 처리: 모든 학생 데이터는 사용자 브라우저 내에서만 처리되어 보안에 안전합니다.
