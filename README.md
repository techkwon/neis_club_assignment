# 나이스 동아리 배정 도우미

4세대 나이스(NEIS) 동아리 부서배정 작업을 CSV 파일을 통해 자동으로 처리하는 크롬 확장 프로그램입니다.

## ✨ 주요 기능

- **CSV 기반 자동 배정**: 학생 데이터(학년, 반, 번호, 이름, 동아리명)가 담긴 CSV 파일을 업로드하면 자동으로 배정
- **스마트 학생 매칭**: 이름 우선 검색, 실패 시 출석번호로 자동 폴백
- **부서구분 매칭**: 동명 동아리 구분 (동아리활동 / 자율동아리 / 청소년단체 / 학교스포츠클럽)
- **자동 정렬**: 학년·반·번호 순 정렬로 드롭다운 변경 최소화
- **일시정지 / 재개 / 정지**: 작업 중 언제든 제어 가능
- **실시간 로그**: 학생별 성공/실패 결과를 실시간 표시
- **작업 결과 요약**: 성공·실패·대기·번호매칭 건수 한눈에 확인
- **화면 꺼짐 방지**: Wake Lock API로 작업 중 절전 모드 방지
- **🔒 온디바이스 처리**: 모든 데이터는 브라우저 내부에서만 처리 (외부 서버 전송 없음)

## 📋 사용 방법

1. 나이스 4세대 → 창의적 체험활동 → 동아리활동관리 → **부서배정** 화면을 엽니다
2. 확장 프로그램 아이콘을 클릭하여 **사이드바**를 엽니다
3. CSV 템플릿을 다운로드하여 학생 데이터를 입력합니다
4. CSV 파일을 업로드하고 **🚀 배정 시작** 버튼을 클릭합니다

## 📄 CSV 양식

| 학년 | 반 | 번호 | 이름 | 동아리명 | 부서구분 |
|---|---|---|---|---|---|
| 1 | 3 | 15 | 홍길동 | 농구1-A | 동아리활동 |

- **번호**: 선택 사항 (이름 매칭 실패 시 폴백용)
- **부서구분**: 선택 사항 (동명 동아리 구분용)

## 🔒 보안

- 모든 CSV 데이터는 브라우저 로컬에서만 처리됩니다
- 외부 서버로 어떠한 데이터도 전송되지 않습니다
- 확장 프로그램을 닫으면 모든 임시 데이터가 삭제됩니다
